Title: Rock Paper Scissors Game

Description:
This project is a simple command-line implementation of the classic Rock, Paper, Scissors game using JavaScript. The program runs in a Node.js environment and allows a user to play against the computer for three rounds.

When the program starts, the user is prompted to input their move ("Rock", "Paper", or "Scissors"). The computer randomly selects its move from the same set of options. The program then determines the winner of each round based on standard game rules:

Rock beats Scissors
Paper beats Rock
Scissors beats Paper

If the user enters an invalid move, the round is automatically awarded to the computer.

After three rounds, the program compares the total scores and declares the overall winner (User, Computer, or Tie).

Features:

Input validation for user moves
Randomized computer move selection
Round-by-round result display
Final game winner determination
Simple and readable game logic

How to Run:

Make sure Node.js is installed.

Run the program using:

node main.js
Enter your move when prompted.

Game Rules:

Valid moves: Rock, Paper, Scissors
Invalid input results in an automatic loss for that round
The game consists of 3 rounds
The player with the highest score at the end wins

File Structure:

main.js — Main program file containing all game logic
